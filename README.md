# Nusrat Nizam Suzana — Academic Portfolio

A professional static academic portfolio website built with HTML, CSS, and JavaScript.

## Deploy on GitHub Pages
1. Create a public GitHub repository named `suzana-nusrat.github.io` or `portfolio`.
2. Upload `index.html`, `styles.css`, and `script.js`.
3. Go to Settings → Pages.
4. Select branch `main` and root folder.
5. Visit the published URL after GitHub Pages finishes deployment.

## Deploy on Netlify
Drag and drop this folder into Netlify, or connect the GitHub repository.

## Deploy on Vercel
Import the GitHub repository into Vercel, or use Vercel CLI from this folder.
